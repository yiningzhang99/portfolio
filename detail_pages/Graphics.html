Splines 2020.10-11

Use OpenGL & C++ to implement splines, rails, dome sky box, and dynamic cameras on roller coaster. Apply openGL Phong shading to rails.

Heightmap 2020.9-10

Use OpenGL & C++ to implement terrains with given Heightmap data or picture. The terrain can change colors, rendering mode (dot, lines, strips, faces), scales, rotations, and positions. Hand-writing Phong shading is implemented.

Snow Deformation (Recovered ver.) 2020.6-8

Use Unity and DirectX3D to implement interactive snow effect in Raise of the Tomb Raider. Use compute shader to calculate dynamic heightmap for snow texture based on object positions; unlit shader to render proper change of vertices; another compute shader to imitate fill of snow effect; hull shader and domain shader to increase efficiency. The snow can be filled.

Interactive Texture 2020.6-8

Use Unity and DirectX3D to implement interactive snow effect in Batman: Arkham Origins. Use a depth camera to catch height information of each vertex and store in heightmap. The snow cannot be filled.
